Como usar:

  • Cole a URL do seu visualizador 3D hospedado no GitHub Pages
  • Ajuste as dimensões conforme necessário ou marque "Layout responsivo"
  • Clique em "Gerar Embed" para criar o código
  • Use "Visualizar" para testar o resultado
  • Copie o código e cole-o em sua plataforma de ensino
  • Os alunos verão apenas o visualizador 3D, sem estas configurações
